# User Guide
This is a regular Duke (Todo list app) with only 1 fancy feature (C-MassOps).
Basic Ui only.
## Features 

### Mass operations
Able to remove multiple tasks at once, and set multiple tasks to done.

## Usage

### `list` - list all tasks

Shows a list of all tasks.

Example of usage: 

`list`

Expected outcome:
`1. [D] [✓] Buy eggs (by: 7pm)
2. [T] [✗] Sell eggs`
