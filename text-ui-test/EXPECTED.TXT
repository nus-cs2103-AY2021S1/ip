How to use Duke:
* Level 8: Formatting Time

* DEVELOPER UPDATES:
* I can now save data! Just terminate the program with "bye" and I'll save the data automatically!
* Do take note of the DATE AND TIME notations when inserting DukeTasks!

AVAILABLE INSTRUCTIONS:
 help - Display Available Instructions
 bye - Terminate Duke
 list - Display current DukeTasks
 done [Task Number] - Complete the specified task number (Specify in numeric format!) Eg: "done 3"
 delete [Task Number] - Deletes the task number (Specify in numeric format!) Eg: "delete 3"
* SEE TASK INPUT INSTRUCTIONS FOR MORE ABOUT TASKS

TASK INPUT INSTRUCTIONS:
IMPORTANT: DO AVOID USING THE VERTICAL BAR "|" IN YOUR TASK INSTRUCTIONS
 todo [Task Description] - Inputs a TODO DukeTask
 deadline [Task Description] /by [Date] - Inputs a DEADLINE DukeTask, using INDICATOR /by
 event [Task Description] /at [Date] - Inputs an EVENT DukeTask, using INDICATOR /at

TO NOTE:
FORMAT FOR DATE: "DD/MM/YYYY hh/mm/ss"
 	DD, MM, YYYY are the date, month and year respectively (IN INTEGERS)
 	hh, mm, ss are the hour (in 24 HOUR NOTATION), minutes and seconds respectively (IN INTEGERS)
