Here are the list of commands you can use! =D
help: displays the list of commands available

list: displays the list of tasks you have

todo *task description*: adds a task without any
date/time attached to it
eg todo read book

deadline *task description* /by *date/time*: adds a
task that needs to be done before a specific date/time
eg deadline return book /by June 5 10pm

event *task description* /at *time*: adds a task that
starts at a specific time and ends at a specific time
eg event meeting /at 2-4pm

done *task number*: marks the task with that number as
done eg done 1

delete *task number*: deletes the task with that number
from the list eg delete 1

bye: ends the session
